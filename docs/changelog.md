## [1.0.0] - 2024-01-31

### Added
- MVP completo da plataforma AgendaEstética
- Sistema multi-tenant com isolamento de dados
- Autenticação Firebase (profissional/cliente)
- Gestão completa de agenda
- Sistema de agendamentos com aprovação
- Portal público para clientes
- Personalização visual (temas)
- Sistema de notificações
- Dashboard profissional
- Gestão de clientes
- Suporte a remarcações
- Deploy automatizado Vercel
- Firebase Firestore como backend
- Testes automatizados (Mocha/Chai)
- Testes manuais documentados

### Technical
- Vanilla JavaScript (ES6+)
- Firebase v9+ modular SDK
- CSS3 com variáveis para temas
- Arquitetura SPA simples
- PWA-ready
- Mobile-first responsive
- Firebase Security Rules
- Firebase Emulator para testes

### Security
- Autenticação obrigatória
- Isolamento de dados por empresa
- Validação client-side e server-side
- HTTPS obrigatório
- Firebase Security Rules ativas
